                                                                                                                                  # 特性

                                                                                                                                  # 层叠行

                                                                                                                                  样式产生冲突，后面的样式会覆盖前面的样式

                                                                                                                                  # 继承性

                                                                                                                                  若子元素未设置相应的样式，则会继承父元素的样式

                                                                                                                                  一般继承与文字有关的样式，如文本颜色，字号

                                                                                                                                  # 优先级

                                                                                                                                  选择器选择器权重
                                                                                                                                  继承,*0,0,0,0
                                                                                                                                  元素选择器0,0,0,1
                                                                                                                                  类选择器，伪类选择，属性选择器0,0,1,0
                                                                                                                                  ID选择器0,1,0,0
                                                                                                                                  行内样式1,0,0,0
                                                                                                                                  !importantmax

                                                                                                                                  复合选择器权重叠加，从左到右依次比较

                                                                                                                                  # 基础选择器

                                                                                                                                  名称形式
                                                                                                                                  标签选择器div、span、p、...
                                                                                                                                  ID选择器#name,#age
                                                                                                                                  类选择器..name，.age
                                                                                                                                  后代选择器ol li
                                                                                                                                  子代选择器ol>li
                                                                                                                                  并集选择器ul,ol
                                                                                                                                  兄弟选择器a~b
                                                                                                                                  相邻选择器a+b
                                                                                                                                  属性选择器
                                                                                                                                  匹配所有属性[xxx]
                                                                                                                                  匹配所有属性，值为value[xxx='value']
                                                                                                                                  匹配所有属性，值含value[xxx*='value']
                                                                                                                                  匹配所有属性，值含value[xxx~='value']
                                                                                                                                  匹配所有属性，值为value开头[xxx^='value']
                                                                                                                                  匹配所有属性，值以value结尾[xxx$='value']

                                                                                                                                  span.a span中类名为a的元素

                                                                                                                                  span .a span后代中类名为a的元素

                                                                                                                                  # 伪类选择器

                                                                                                                                  选择器描述
                                                                                                                                  :link未访问
                                                                                                                                  :visited已访问
                                                                                                                                  :hover鼠标指针移动在元素上
                                                                                                                                  :active鼠标按下未松开时使用
                                                                                                                                  :focus指定元素获得焦点时使用
                                                                                                                                  :enabled指定元素处于可用状态时
                                                                                                                                  :disabled指定元素处于禁用状态时
                                                                                                                                  :checked指定radio/checkbox处于选取状态
                                                                                                                                  :default当页面打开时默认处于选取状态的radio/checkbox的状态(即使状态改变样式仍然有效)
                                                                                                                                  :read-only指定元素处于只读状态
                                                                                                                                  :read-write指定元素处于读写状态
                                                                                                                                  :indeterminate当页面打开时一组单选框中没有任何一个单选框被选中时整组单选框的样式(状态改变则样式取消)
                                                                                                                                  :valid当元素通过如required/pattern等属性检查或元素内容符合元素规定样式时
                                                                                                                                  :invalid当元素不能通过如required/pattern等属性检查时或元素内容不符合元素规定格式时
                                                                                                                                  :required已经指定required属性的input/select/textarea元素
                                                                                                                                  :optional未指定required属性的input/select/textarea元素
                                                                                                                                  :in-range实际输入值在范围内
                                                                                                                                  :out-of-range实际输入值在范围外
                                                                                                                                  :root文档根元素
                                                                                                                                  :no(x)选择非x的元素
                                                                                                                                  :target当前激活的锚点
                                                                                                                                  :empty无子元素(包含文本节点)
                                                                                                                                  :first-child首个子元素
                                                                                                                                  :last-child尾部子元素
                                                                                                                                  :only-child仅有一个子元素
                                                                                                                                  :nth-last-child(n)倒数第n个子元素
                                                                                                                                  :nth-child(n)n：本身，所有子元素
                                                                                                                                  n：数字，第n个子元素
                                                                                                                                  n：even，偶数子元素
                                                                                                                                  n：odd，奇数子元素
                                                                                                                                  n：an+b，n初始值为0
                                                                                                                                  :nth-of-type(n)同:nth-child(n)
                                                                                                                                  :nth-last-of-type同:nth-last-child

                                                                                                                                  1.target:nth-child(n) 无论子元素类型是否一样都是按照从头开始排序,先找到第n个子元素再判断该子元素是否与target匹配

                                                                                                                                  2.target:nth-of-type(n) 根据类型进行排序，先对target进行排序再找第几个子元素

                                                                                                                                  元素模式元素排列设置样式默认宽度包含关系常见标签
                                                                                                                                  块级元素独占一行可设置宽高默认容器100%任意标签div
                                                                                                                                  行内元素一行多个不可直接设置内容宽度文本或其他行内元素span a
                                                                                                                                  行内块元素一行多个可设置宽高内容宽度img td input p

display: inline
display: inline-block

display: run-in

                                                                                                                                  # 可视

                                                                                                                                  语法说明
                                                                                                                                  display:none元素隐藏并且不再占有原来位置
                                                                                                                                  display:block元素可视
                                                                                                                                  visibility:hidden元素隐藏且占有原来位置
                                                                                                                                  visibility:visible元素可视
                                                                                                                                  overflow:visible溢出部分显示
                                                                                                                                  overflow:hidden溢出部分隐藏
                                                                                                                                  overflow:scroll溢出与否都显示滚动条
                                                                                                                                  overflow:auto溢出显示滚动条，不溢出不显示

                                                                                                                                  值个数描述

                                                                                                                                  # 边距

                                                                                                                                  属性描述
                                                                                                                                  margin外边距
                                                                                                                                  padding内边距
                                                                                                                                  margin: 1px 1px 1px 1px
padding: 1px 1px 1px 1px

                                                                                                                                  外边距合并

                                                                                                                                  对于两个嵌套关系的块元素，父元素与子元素都有外边距，此时父元素会塌陷较大的外边距(仅仅针对标准流)

                                                                                                                                  解决方案：
1.为父元素添加边框
2.为父元素添加内边距
3.为父元素添加overflow:hidden

                                                                                                                                  # 换行

                                                                                                                                  word-break

                                                                                                                                  属性值描述
                                                                                                                                  normal(默认)单词内不断行,标点符号不会在句首
                                                                                                                                  break-all单词内任意位置可以断行,允许标点在句首
                                                                                                                                  keep-all单词内任意位置不断行,只能在半角空格或连字符或标点符号处换行，文字与文字之间不换行

                                                                                                                                  white-space

                                                                                                                                  属性值换行符空格与制表符(Tab)文字换行
                                                                                                                                  normal合并合并换行
                                                                                                                                  nowrap合并合并不换行(强制一行显示)
                                                                                                                                  pre保留保留不换行
                                                                                                                                  pre-wrap保留保留换行
                                                                                                                                  pre-line保留合并换行

                                                                                                                                  word-wrap

                                                                                                                                  URL地址自动换行，默认不会在URL中换行，它被当成长单词对待 当超过浏览器窗口大小时，会出现滚动条

                                                                                                                                  word-wrap:break-word //允许URL中换行

                                                                                                                                  # 渐变

                                                                                                                                  # 线性渐变

                                                                                                                                  渐变方向

                                                                                                                                  background-image:linea-gradient(direction,color1,color2,..)

                                                                                                                                  to bottom       上到下(默认)
to top          下到上 
to right        左到右
to left         右到左
to bottom right 左上到右下
to bottom left  右上到左下

                                                                                                                                  渐变角度(顺时针)

                                                                                                                                  background-image:linear-gradient(XXXdeg,color1,color2,..)

                                                                                                                                  渐变百分比(纯色块)

                                                                                                                                  background-image: linear-gradient(XXX,color1 X%,color2 X%,..)

                                                                                                                                  重复渐变

                                                                                                                                  background-image: repeating-linear-gradient(to right,red ,yellow 50%)

                                                                                                                                  # 径向渐变

                                                                                                                                  background-image: radial-gradient(shape style at x y,red,color1,color2,..)
                                                                                                                                  属性属性值说明
                                                                                                                                  shape(可选)circle圆
                                                                                                                                  ellipse椭圆
                                                                                                                                  style(可选)farthest-side可渐变到离渐变起点最远的边
                                                                                                                                  closest-corner可渐变到离渐变起点最近的一个角
                                                                                                                                  farthest-corner可渐变到离渐变起点最远的一个角
                                                                                                                                  x y(可选，渐变起点)
                                                                                                                                  center center元素中心点向外扩散
                                                                                                                                  left top元素左上角向外扩散
                                                                                                                                  center top元素顶部中央向外扩散
                                                                                                                                  right top元素右上角向外扩散
                                                                                                                                  right center元素右端中央向外扩散
                                                                                                                                  right bottom元素右下角向外扩散
                                                                                                                                  center bottom元素底部中央向外扩散
                                                                                                                                  left bottom元素左上角向外扩散
                                                                                                                                  坐标值指定标点向外扩散

                                                                                                                                  重复渐变

                                                                                                                                  background-image: repeating-radial-gradient(circle,red,red 25%,yellow 25%,yellow 50%,green 50%,green 75%) //与线性渐变一样，可以不均匀，也可以纯色

                                                                                                                                  # 圆锥渐变

                                                                                                                                  默认角度0，锥心为中心点

                                                                                                                                   background-image: conic-gradient(blue,white)

                                                                                                                                  改变起始角度

                                                                                                                                  background-image: conic-gradient(from 90deg,blue,white)

                                                                                                                                  改变锥心置

                                                                                                                                  background-image: conic-gradient(from 90deg at 150px 100px,blue,white)

                                                                                                                                  设置颜色起始位置，接受百分比和角度

                                                                                                                                  background-image: conic-gradient(from 45deg,yellow 20%,red 20%)

                                                                                                                                  重复渐变

                                                                                                                                   background-image: repeating-conic-gradient(yellow 0 36deg,red 36deg 72deg)

                                                                                                                                  # 背景

                                                                                                                                  background:颜色 地址 平铺 滚动 位置

                                                                                                                                  属性描述
                                                                                                                                  backgorund-color支持#,rgba
                                                                                                                                  background-image:url()多背景，逗号分隔，z-index控制层级

                                                                                                                                  # 平铺

                                                                                                                                  属性 （background-repeat）描述
                                                                                                                                  no-repeat不平埔
                                                                                                                                  repeat-x横向
                                                                                                                                  repeat-y纵向
                                                                                                                                  round平铺(可能会压缩图像，以防止边缘图像被裁剪)
                                                                                                                                  repeat平铺,覆盖整个背景区域(默认，边缘图像可能被剪切)
                                                                                                                                  space平铺(图像之间可能会增加空白，以防止边缘图像被裁剪)

                                                                                                                                  # 滚动

                                                                                                                                  设置背景图是否跟随页面滚动

                                                                                                                                  属性 （background-attachment）描述
                                                                                                                                  scroll跟随
                                                                                                                                  fixed不跟随
                                                                                                                                  local跟随元素内容滚动

                                                                                                                                  # 浮动

                                                                                                                                  # 浮动原理

                                                                                                                                  • 浮动的最初目的是形成文字的环绕效果即压不住文字，而固定定位与绝对定位则压住所有
                                                                                                                                  • 浮动元素不再占有原来位置
                                                                                                                                  • 多个元素设置浮动，则一行内显示并且顶端对齐排列，父级元素无法容纳则另起一行
                                                                                                                                  • 任何元素都能够使用浮动，则该元素将具有块元素的性质,并且在同一行如果父元素宽度允许
                                                                                                                                  • 多个块级元素纵向排列标准流，多个块级元素横向排列浮动

                                                                                                                                  # 引发问题

                                                                                                                                  • 父元素的高度无法被撑开，影响兄弟元素
                                                                                                                                  • 如果浮动元素后紧跟着非浮动元素，则非浮动元素也会随之浮动起来，但是并不会与前一个浮动元素并列而是在标准流文档时的位置
                                                                                                                                  • 浮动元素的同级元素也需要浮动否则影响页面的结构

                                                                                                                                  # 清除浮动

                                                                                                                                  【原因】父级元素不便设置高度,需要子元素撑开,自动检测高度,但是子元素设置浮动不占空间,则父级元素检测到的高度为零

                                                                                                                                  • 额外标签法(闭合浮动)
                                                                                                                                  父级元素最末端添加<div class="clear"></div>(必须是块级元素)

.clear {
    clear:both
}
                                                                                                                                  • 父级元素添加overflow
                                                                                                                                  overeflow:hidden/auto/scorll
                                                                                                                                  • 父级元素添加伪元素
                                                                                                                                  .clearfix:after {
    content:"";
    display:block;
    height:0;
    clear:both;
    visibility:hidden;
}

.clearfix {
    *zoom:1;
}
                                                                                                                                  • 父级元素添加双伪元素
                                                                                                                                  .clearfix::before,.clearfix::after {
    content:"";
    display:table; 
}
.clearfix::after {
    clear:both;

    *zoom:1;

                                                                                                                                  # 格式化

                                                                                                                                  # BFC

                                                                                                                                  块级格式上下文，它是独立的渲染区域，其中的元素布局不受外界的影响，也不会影响外界布局

                                                                                                                                  规则：

                                                                                                                                  • 内部box垂直方向，排列放置
                                                                                                                                  • 同一个BFC相邻的box的垂直方向margin会发生折叠，折叠大的值
                                                                                                                                  • 从左向右格式化时，内部box的左边与包含块左边相接触（包含块padding为0），即使存在浮动也是如此
                                                                                                                                  • BFC区域不与float区域重叠
                                                                                                                                  • 计算BFC高度时，浮动元素参与计算

                                                                                                                                  创建BFC：

                                                                                                                                  • float不为none
                                                                                                                                  • position不为static或relative
                                                                                                                                  • display为inline-block、table-cell、table-caption、flex、inline-flex
                                                                                                                                  • overflow不为visible

                                                                                                                                  BFC作用：

                                                                                                                                  • 避免margin重叠
                                                                                                                                  • 自适应两栏布局
                                                                                                                                  • 清楚浮动撑开父元素

                                                                                                                                  # IFC

                                                                                                                                  内联格式上下文，（不属于块元素的所有可是元素都是内联元素）

                                                                                                                                  规则：

                                                                                                                                  • 子元素横向排列，垂直方向起点为包含块顶部
                                                                                                                                  • 子元素只会计算横向样式，垂直方向样式不计算（padding、border、margin）
                                                                                                                                  • 能把一行上的框都完全包含进去的矩形区域称为该行的行框，行框高度为最高元素的高度，行框的宽度由包含块与其中的浮动元素决定
                                                                                                                                  • 行框宽度大于包含块时会换行

                                                                                                                                  创建IFC

                                                                                                                                  块级元素只包含内联元素

                                                                                                                                  IFC作用：

                                                                                                                                  • 上下间距不生效
                                                                                                                                  • 水平居中，当一个块要在环境中水平居中时，将其设置为inline-block则会在外层产生IFC，通过text-align：center可实现
                                                                                                                                  • 垂直居中，创建IFC，其中一个元素撑开父元素高度，父元素设置vertical-align：middle，其他行内元素可在此父元素下垂直居中

                                                                                                                                  # 滤镜

                                                                                                                                  filter: blur()  | grauscale() |
                                                                                                                                  特效描述（一般取值是百分比）
                                                                                                                                  grayscale彩色图转换为灰色图
                                                                                                                                  sepia彩色图添加棕褐色色调，呈现老照片效果
                                                                                                                                  saturate增强彩色图饱和度，使色彩鲜明，呈现海报或卡通效果
                                                                                                                                  hue-rotate产生特殊视觉效果，红色到紫色颜色环，将图像颜色沿光环旋转一定角度(deg)
                                                                                                                                  invert图像颜色翻转
                                                                                                                                  opacity图像透明度
                                                                                                                                  contrast调整图像对比度，200%对比度为原图两倍
                                                                                                                                  blur模糊，为图像添加一层玻璃遮罩，产生烟雾效果，(像素值)
                                                                                                                                  drop-shadow为图像添加阴影效果，多个可选参数横向偏移，纵向偏移，模糊半径，阴影颜色

                                                                                                                                  # 容器

                                                                                                                                  常用于图片、视频使其充满容器

                                                                                                                                  object-fit: fill | contain | cover | scale-down | none
                                                                                                                                  值描述
                                                                                                                                  fill默认，不保证原有比例，内容来神填充容器内容
                                                                                                                                  contain保持原有比例，内容缩放
                                                                                                                                  cover保持原有比例，但部分内容可能被剪切
                                                                                                                                  none保持原有长度宽度
                                                                                                                                  scale-down保持原有比例，内容的尺寸与 none 或 contain 中的一个相同，取决于它们两个之间谁得到的对象尺寸会更小一些
