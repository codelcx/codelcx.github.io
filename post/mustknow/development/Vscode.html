                  # 标签生成

                  标签名+tab  			  // 生成标签

标签名*数字+tab 			//生成多个相同标签

标签名>标签名+tab			//父子级关系

标签名+标签名+tab   		//兄弟级关系

标签名.类名+tab     		 //生成含类名标签(默认div)

标签名#类名+tab			//生成含ID标签

标签名.类名$*数字+tab     //生成有序类名标签  ，$代表递增

标签名{XXX}+tab  		  //生成含默认字符标签

                  # 快捷按键

                  按键组合描述
                  ctrl+f查找
                  ctrl+h替换
                  ctrl+p文件跳转
                  ctrl+tab切换到最近打开的文件
                  ctrl+home跳转到页头
                  ctrl+end跳转到页尾
                  ctrl+enter当前行下插入新一行
                  ctrl+shift+enter当前行上插入新一行
                  shift+del删除光标所在行
                  alt+shift+click同列多光标操作
                  alt+up/down将光标所处行按上下移动
                  ctrl+shift+left/right光标定位到单词后，可配合多光标使用
                  ctrl+f2选中所有当前选中词汇
                  ctrl+ =+/-放大/缩小
                  ctrl+f4/w关闭当前编辑窗口
                  alt+left/right切换编辑器窗口
